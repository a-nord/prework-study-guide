HTML

  • Hypertext Markup Language (HTML), to give the content structure.
  • Declare what type of document the browser should be and that html should be expected on the page.
    • For example, !DOCTYPE html, which is what we are using for this website.
  • The head element contains information, metadata, about the webpage.
    • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
    • The title element of a webpage (tab) is contained in the head element.
    • The style element defines the CSS styles associated with a website.
    • The base element defines the base URL for a webpage.
    • The link element connects an external resource to the HTML document.
    • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
    • The script element embeds client-side scripts in the HTML document.
  • The body element represents the visible content shown to the user.
    • The body is an example of a semantic HTML element, which is the name of the element describes its content and the role it plays on a webpage.
    • Child elements are elements nested inside other elements.
    • The parent element holds all the children. The "parent/child" language is often used by developers to explain the relationship among elements in HTML.
    • When working with visible HTML elements, the content is contained between an opening tag and a /closing tag.
    • h1, h2, h3, etc. elements represents the level of heading a given text block represents.
    • p element represents a paragraph or block of text.
    • ul,ol, and li elements represent unordered lists, ordered lists, and list items. Could be bulleted or numbered.
    • img element contains info about images that are displayed on the webpage. It can contain various attribute; the src attribute is the most important because it defines the location of the image.
    • br element creates a line of empty space, or a line break, between two blocks of content.
    • hyperlink, the a element (which stands for anchor) creates links to the same webpage or other webpages. The href attribute points to the URL for the link
  • The HTML attribute defines additional information about an element. This is always defined in the beginning element.

CSS

  • Cascading Style Sheets (CSS), to give the content style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript (JS), to handle logic and allow users to interact with a website.
